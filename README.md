# 🍽️ Ohmyfood

Application web **mobile-first** dédiée à la découverte de restaurants gastronomiques, avec animations fluides en CSS et SASS.

---

## 🚀 Description

**Ohmyfood** est une application front-end qui permet aux utilisateurs de :

* Découvrir des restaurants gastronomiques
* Consulter les menus détaillés
* Réserver une table
* Précomposer leurs plats à l’avance

L’objectif principal est d’offrir une **expérience utilisateur fluide, rapide et moderne**, notamment grâce à des animations CSS avancées.

---

## 🎯 Objectifs du projet

* Créer une interface **mobile-first**
* Implémenter des **animations fluides en CSS / SASS**
* Améliorer l’expérience utilisateur avec des interactions dynamiques
* Structurer un projet front-end propre et maintenable

---

## 🛠️ Technologies utilisées

* HTML5
* CSS3
* SASS (SCSS)
* Animations CSS (keyframes, transitions)

---

## ✨ Fonctionnalités

* 📱 Design responsive (mobile-first)
* 🎨 Animations fluides (chargement, hover, transitions)
* 🍴 Présentation de menus interactifs
* ❤️ Sélection de plats avec effets visuels
* ⚡ Interface rapide et optimisée
